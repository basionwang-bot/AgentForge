# 🔒 私有定制课程(企业 / 团队)

> 此目录用于**企业或团队的私有定制课程**,**不纳入公开课程地图**(`课程地图.md`),也不被 `manifest.json` 收录。
>
> 放在这里的课,通常是:企业**内部流程课**、**内部工具课**、行业私有 know-how——不适合公开,但同样需要 agent 学会。

## 规矩(私有不等于放水)

- **同一套质量闸门**:私有课一样必须通过 [`../../出课标准.md`](../../出课标准.md) 的验收清单——严进严出不因私有而打折。
- **同一套考官协议**:私有课的毕业测验,一样要真执行、留证据、过[独立考官](../../校规.md)。
- **同一套安家流程**:学完的技能卡按 [`../../毕业安家.md`](../../毕业安家.md) 落到 agent 工作的全局技能目录。

## 怎么用

1. 在本目录下按学院/编号建私有课文件(如 `private/P01-内部审批流程.md`),套用出课标准的"解剖图"结构。
2. 私有课**不要**登记进公开 `课程地图.md`;团队内部另建索引即可。
3. 给内部 agent 的入职/上学指令里,把选课范围指向 `courses/private/`。

> 想为团队系统化定制?见 [`../../企业入职指南.md`](../../企业入职指南.md)。
