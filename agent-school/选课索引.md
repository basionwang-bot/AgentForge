## 开方三原则

1. **少而精**:只挑 2–3 门最对口的,别贪多——入职是"先上岗",不是"读完整本书"。
2. **底线先行**:无论办什么,**基础 01「先探索再动手」必修**,它决定 agent 会不会乱来。
3. **任务驱动**:挑课的唯一标准是"能不能帮主人把第一步面谈里那件事办成",不是"课好不好"。

> 修完这几门 → 沉淀技能卡 → 按 [毕业安家.md](毕业安家.md) 装到主人电脑 → 立刻上岗交付。
