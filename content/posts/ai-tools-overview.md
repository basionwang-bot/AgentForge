## 🧭 你先对号入座（A / B / C / D）

| 类型 | 你是谁 | 该重点看 |
|------|--------|----------|
| **A** | 完全小白，只想"用 AI 帮我搞定具体小事" | 通用对话(1) + AI 搜索(2) + AI 写作(3) + 翻译(9) |
| **B** | 职场人，想提效（报告、表格、汇报、PPT） | 通用对话(1) + PPT 办公(8) + 文档阅读(13) + 编程辅助(10·选学) |
| **C** | 想做副业/搞钱（公众号、小红书、抖音） | 写作(3) + 图片(4) + 视频(5) + 数字人(6) + 配音(7) + 智能体(12) |
| **D** | 传统行业老板/员工，想给业务加 AI | 通用对话(1) + 智能体平台(12) + 行业应用(14) + 数字人(6，做客服) |

> 不知道选哪类？**先从分类 1（通用对话）开始，这是所有人都用得到的"万能瑞士军刀"。**

## 一、💬 通用对话/大模型（所有人的入门第一步）

> 这是 AI 时代的"百度 + Word + 顾问"，学一个就够日常用。**不要纠结哪个最好，选一个开始用。**

### 1. DeepSeek 🇨🇳
- **官网**：chat.deepseek.com
- **定位**：国产之光，中文场景效果最好，**完全免费**
- **适合**：A / B / C / D（全部）
- **收费**：网页 / App **完全免费**；API 按量付费，价格白菜
- **上手难度**：★（下载即用）
- **推荐场景**：写文案、写报告、查资料、写代码、做学习辅导
- **避坑**：深度思考（R1）模式比普通模式聪明很多，**遇到难题记得开 R1**

### 2. 豆包（字节跳动）🇨🇳
- **官网**：doubao.com / App 商店搜"豆包"
- **定位**：字节家的 AI，**手机端体验最丝滑**，带语音、带摄像头实时识别
- **适合**：A / B / C / D
- **收费**：免费
- **上手难度**：★
- **推荐场景**：手机随时问、语音聊天、拍照识别（看说明书、扫菜单翻译）
- **避坑**：web 端能力比 App 强，严肃任务用网页

### 3. Kimi（月之暗面）🇨🇳
- **官网**：kimi.com
- **定位**：**长文档处理之王**，可以一次扔进去几十万字让它读
- **适合**：B / C / D（职场人/做内容/老板）
- **收费**：免费
- **上手难度**：★
- **推荐场景**：**整本书让它总结、几十页 PDF 让它提炼要点、合同让它找风险点**
- **避坑**：它最强的是"读得多"，不是"想得深"，复杂推理还是用 DeepSeek

### 4. 通义千问（阿里）🇨🇳
- **官网**：tongyi.aliyun.com
- **定位**：阿里家的 AI，**生态最全**（电商/办公/编程都覆盖）
- **适合**：B / D（企业/电商相关）
- **收费**：免费
- **上手难度**：★
- **推荐场景**：跟阿里系产品（钉钉、飞猪、1688）联动、电商相关任务
- **避坑**：个人使用不一定优先选，看你需要不需要它的生态

### 5. 智谱清言（GLM）🇨🇳
- **官网**：chatglm.cn
- **定位**：清华系，**严肃任务/学术场景表现好**
- **适合**：B（职场）、学生
- **收费**：免费 + 付费会员
- **推荐场景**：论文辅助、严肃写作、专业问题
- **避坑**：气质偏"学院派"，闲聊不如豆包活泼

### 6. ChatGPT（OpenAI）🌍
- **官网**：chatgpt.com（国内需自备网络）
- **定位**：鼻祖，综合能力强，**插件生态最丰富**
- **适合**：B / C（有点基础的）
- **收费**：免费版可用；Plus 20 美金/月；Pro 200 美金/月
- **上手难度**：★★（网络问题、付费问题）
- **推荐场景**：复杂任务、多模态、用海外资源
- **避坑**：**国内学员先用 DeepSeek + 豆包**，这个不是必需

### 7. Claude（Anthropic）🌍
- **官网**：claude.ai（国内需自备网络）
- **定位**：**写作和代码界公认天花板**，文风克制有质感
- **适合**：B / C（写文案、写代码）
- **收费**：免费版有限额；Pro 20 美金/月；Max 100/200 美金/月
- **上手难度**：★★
- **推荐场景**：写公众号、写小说、写代码、改稿子
- **避坑**：中文能力很好但偶尔会冒英文，文案场景做最后一遍校对

### 8. Gemini（Google）🌍
- **官网**：gemini.google.com
- **定位**：谷歌家的 AI，**搜索整合好、免费版能力强**
- **适合**：B（国际化场景）
- **收费**：免费版很能打；Advanced 20 美金/月
- **避坑**：国内访问需要网络

### 9. Grok（xAI）🌍
- **官网**：x.ai / grok.com
- **定位**：马斯克家的，**实时性强**（直接拉 X/推特数据）、风格不羁
- **适合**：C（关注热点、做内容）
- **收费**：X Premium 订户免费用
- **避坑**：回答风格"皮"，正式场景慎用

## 三、✍️ AI 写作

### 13. 写作直接用通用大模型（DeepSeek / Kimi / Claude）
- **建议**：**90% 的写作场景，通用大模型直接搞定，不需要单独的"写作工具"**
- 用法：把"写什么 + 给谁看 + 什么风格 + 多少字"讲清楚

### 14. 火山写作 / 讯飞写作 / 秘塔写作猫 🇨🇳
- **定位**：聚焦"中文文章润色、查重、降重"
- **适合**：学生（论文）、自媒体作者
- **收费**：免费试用 + 付费

### 15. 笔灵 AI / Notion AI 🇨🇳🌍
- **定位**：模板化写作（简历、周报、邮件、营销文案）
- **适合**：A / B（不知道从哪下笔的人）
- **收费**：免费额度 + 会员

## 四、🎨 AI 图片生成

### 16. 即梦 AI（字节）🇨🇳
- **官网**：jimeng.jianying.com
- **定位**：**国产图像生成第一梯队**，中文理解好、人像/国风强
- **适合**：C（做内容）、A（玩）
- **收费**：免费有积分、付费会员去水印/加速
- **推荐场景**：小红书封面、公众号配图、海报、人像
- **避坑**：**写中文 prompt 即可，不用学英文咒语**

### 17. 可灵图像（快手）🇨🇳
- **官网**：klingai.com
- **定位**：**人像和真实感场景的天花板**，适合做"以假乱真"的图
- **适合**：C
- **收费**：免费积分 + 会员
- **推荐场景**：产品图、人像写真、电商图

### 18. 通义万相（阿里）/ 文心一格（百度）🇨🇳
- **定位**：大厂出品，稳定够用
- **建议**：**主用即梦+可灵就够了**，这两个备选

### 19. Midjourney 🌍
- **官网**：midjourney.com
- **定位**：**全球公认艺术效果天花板**，审美在线
- **适合**：C（对画面有追求的内容创作者）
- **收费**：10 / 30 / 60 / 120 美金/月，**没有免费版**
- **上手难度**：★★（需要英文 prompt、需要网络）
- **避坑**：**中文学员先用即梦，真有审美追求再上 MJ**

### 20. Flux / Stable Diffusion 🌍
- **定位**：开源派，可以本地部署、可以炼丹
- **适合**：技术控、想做差异化的设计师
- **收费**：开源免费，但需要硬件/会折腾

### 21. Recraft 🌍
- **官网**：recraft.ai
- **定位**：**矢量图/图标/UI 设计专精**
- **适合**：设计师、做品牌的
- **收费**：免费额度 + 会员
