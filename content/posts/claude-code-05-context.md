---

## 先回头看看那张"桌子"

还记得开篇那个画面吗？Claude 干活的时候，脑子里就摆着一张**大小有限的写字台**。你俩聊的每句话、它读过的每个文件、跑命令吐出来的每段结果，全都一股脑摊在这张桌子上。东西堆得越来越满，桌面越来越乱——到最后它就开始忘事、抓不住重点、犯些低级得让你想叹气的错。

道理很朴素：桌子就这么大，堆爆了，再聪明的脑子也翻不动。**这一篇，从头到尾就教你一件事：怎么把这张桌子收拾利索。** 学会了，你一只脚就迈进高手的门槛了。

---

## 一、纠偏与回滚：随时把它拽回正道

### `Esc` —— 随时喊停

发现它正闷头往错路上狂奔？别干等着，按 **`Esc`** 立刻把它叫停。**桌上的东西全都原样保留**，你只要补一句话重新给它指个方向就行，完全不用推倒重来。

### `Esc + Esc` 或 `/rewind` —— 打开"后悔药"菜单

连着按两下 `Esc`（或者直接输入 `/rewind`），会弹出一个回滚菜单。你可以挑：

- **只把对话退回去**（代码原样保留）
- **只把代码退回去**（对话原样保留）
- **对话和代码一起退**

它每收到你一条提示词，都会**悄悄存一个存档点**，而且换了会话也还在——说白了，这就是一颗随时能嗑的"后悔药"。

> ⚠️ **重要提醒：** 这颗后悔药只管 **Claude 自己动过的地方**，它**顶替不了 git**。你自己手动改的东西、或者更正经的版本管理，那还得靠 git。记住这个分工：`/rewind` 当"小范围一键撤销"使，git 才是"正经的版本管理"。

### "撤销刚才的改动" —— 张嘴说就行

还有最省事的一招：直接打字让它自己退回去，连按键都省了。

```text
撤销你刚才对登录逻辑的改动
```

## 二、🌟 核心规律（本模块最值钱的一段，请反复读）

> **同一个问题，纠正它两回还是不对，就别再纠第三回了。**

为啥？因为到了这步，**它那张桌子已经被一摞失败的尝试给污染透了。** 桌面上堆满了"试过 A 没用、又试 B 没用、再试 C 还是没用"的废纸团，它越翻越乱、越想越糊涂，你越纠它越糟，俩人一块儿钻进死胡同。

正确的做法干脆利落，三步：

1. **`/clear` 一把扫干净**——把那张乱成一团的桌子整个掀平，重新铺张白纸。
2. 把刚才两回失败**换来的教训**，揉进一条**写得更到位的开场提示词**里。
3. 拿着这条新提示词，**另开一个干干净净的会话**，从头来过。

> **一个干净的会话 + 一条更好的提示词，几乎每次都能赢过一个堆满了纠错废纸的超长会话。**

这是新手最反直觉、却也最值钱的一课：**有时候"另起炉灶"远比"硬着头皮死磕"快得多。** 别心疼前面那段对话——它早就不是资产，是拖你后腿的负担了。

## 三、日常记忆操作（高频命令）

| 命令 | 干什么 | 什么时候用 |
|------|--------|------------|
| `/clear` | 把整张桌子扫空 | **两个不搭界的任务之间，必扫。** 这是所有好习惯里的头一条。 |
| `/compact 聚焦 API 改动` | 定向收纳，告诉它哪些别收走 | 活还没干完但桌子快满了，收拾一下接着干 |
| `/btw` | 顺手问个小问题，不进对话历史 | 临时想插问点别的，又不想弄脏当前这摊活 |
| `/rename` | 给会话起个名（如 `oauth-migration`） | 把会话当 git 分支一样管理 |

几点补充说明：

- **在 CLAUDE.md 里事先写好收纳规矩**：比如写上一句"收拾桌子时，永远别动改动文件清单和测试命令"，这样往后每次 `/compact`，都不会把关键东西连带打包收走。（CLAUDE.md 到底是个啥？模块六会专门讲。）
- **`/btw` 的妙处**：它把答案丢在一个浮层里，你看完即走，**一点都不占当前这摊活的地方**。想随手查个小东西、又怕"弄脏桌面"的时候，这就是神器。
- **把会话当分支管**：用 `/rename` 起好名之后，用 `claude --continue` 接上最近那一次，用 `claude --resume` 从列表里挑一个继续。手头好几个活同时开工的时候，这套管法能救你的命。

## 四、🔍 子代理（Subagent）调研法：翻一大堆资料，却不弄脏你的桌子

这是收拾桌子的所有招数里最优雅的一招，新手务必学会。

**先说说让人头疼的场景：** 你想让它"调研一下我们的认证系统到底怎么处理 token 刷新"。可"调研"这两个字背后，意味着它得吭哧吭哧翻开一大堆文件——等它翻完，你主对话那张桌子早被几十个文件原文塞得满满当当，挤得你后面想干点正事都没地方下手了。

**破解的办法：派个助手去隔壁房间翻。**

```text
用子代理去调研我们的认证系统怎么处理 token 刷新，
以及有没有现成的 OAuth 工具类可以复用。
```

这个助手（子代理）会**在它自己那张独立的桌子上**埋头翻那一大堆文件，翻完了**只揣着一句结论回来跟你汇报**——那几十个文件的原文，全都留在它自己那间屋里，**一张纸都不往你的主桌子上搬**。

> 💡 一句话记住它：**让人去隔壁房间把资料翻完，只回来跟你说个结论。** 你这张桌子自始至终干干净净，一门心思盯着手里的活就行。

## ✍️ 学员练习

盯着下面这三个习惯，刻意练满一整周：

1. **每换一个不搭界的任务，先 `/clear`。** 逼自己练成下意识的肌肉记忆，手比脑子先动。
2. **下次又掉进"纠了第二回还是不对"的坑里时，咬住牙别纠第三回**——`/clear` 扫掉，把教训写成更好的提示词重开一局。顺手记一笔：这回重开，是不是真比硬磕来得快。
3. **下次要做"大范围翻资料"的活时，派子代理去翻**，亲身体会一下主对话那张桌子始终清爽是个什么滋味。
